# WebBrief

Webapp fullstack para gestión de briefs web.

## Stack

- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Base de datos:** SQLite

## Estructura

```
webbrief/
  frontend/   # React + Vite
  backend/    # Node.js + Express + SQLite
```

## Inicio rápido

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
