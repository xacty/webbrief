# Roadmap: WeBrief

## Overview

WeBrief es una plataforma web para gestionar briefs de proyectos creativos: editor TipTap con secciones, comentarios anclados estilo Google Docs, modos de handoff/preview/share, y flujos de aprobación. Roadmap tracked via GSD desde el bootstrap del milestone v1.0 (UI System Refactor).
